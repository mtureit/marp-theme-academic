# Marp Academic Theme テンプレートガイド

このドキュメントは、`presentation_template.md` を使用して新しいプレゼンテーションを作成するためのガイドです。

## 1. 基本的な使い方

1.  `presentation_template.md` をコピーし、新しいファイル名（例: `my_presentation.md`）に変更します。
2.  ファイル内の `[角括弧]` で囲まれたプレースホルダーを、ご自身の研究内容に合わせて書き換えてください。
3.  `marp` コマンドやVS Codeの拡張機能を使って、スライドをプレビュー・出力します。

## 2. デザインパターン集

以下に、`SITA2025_presentation_松浦.md` で使用された主なデザインパターンをまとめました。コピー＆ペーストしてご自身のスライドに活用してください。

---

### 2.1. 2カラムレイアウト

左右にコンテンツを並べて比較・対比させる場合に使用します。

**コード:**
```markdown
<div class="two-columns">
<div class="column">

### 従来のアプローチ ❌
- [従来手法の問題点1]
- [従来手法の問題点2]

</div>
<div class="column">

### 本研究のアプローチ ✅
- <span class="marker">[本研究の優位性1]</span>
- <span class="marker">[本研究の優位性2]</span>

</div>
</div>
```

---

### 2.2. 強調ボックス (bubble)

スライド内で最も伝えたいメッセージを、吹き出し風のボックスで強調します。

**コード:**
```markdown
<div class="bubble">
[このスライドで最も伝えたいメッセージ]
</div>
```

---

### 2.3. テキストハイライト (marker)

文章中の一部をマーカーでハイライトし、注意を引きます。

**コード:**
```markdown
- <span class="marker">[本研究の優位性1]</span>
```

---

### 2.4. アイコン・絵文字

セクションタイトルや箇条書きの項目に絵文字を追加することで、視覚的な分かりやすさを向上させます。

**コード:**
```markdown
## 発表の流れ

1. **研究背景** 🔍 - [背景の要点]
2. **提案手法** ⚙️ - [手法の要点]
3. **実験方法** 📊 - [実験の要点]
```

---

### 2.5. 数式 (KaTeX)

`math: katex` をフロントマターで指定しているため、LaTeX形式で数式を記述できます。

**コード:**
```markdown
**Accept率** $A_i$
$$
A_i = \frac{N_i}{T_i \times 3}
$$
```

---

### 2.6. テーブル（表）

Markdownの標準的なテーブル記法で表を作成できます。コロン `:` を使って、カラムの文字揃え（左、中央、右）を指定できます。

**コード:**
```markdown
| [項目1] | [項目2] | [項目3] |
|:---|:---:|---:|
| [データA1] | [データA2] | [データA3] |
| [データB1] | [データB2] | [データB3] |
```

---

## 3. テーマについて

このテンプレートは `theme: academic-tml` を使用しています。他のテーマ（`academic-tcue`, `academic-wsd`）に変更する場合は、フロントマターの `theme` の値を書き換えてください。

```yaml
---
marp: true
theme: academic-tcue  # ここを変更
paginate: true
math: katex
---
